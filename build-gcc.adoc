なんで〜？ 

とりあえずprintfデバッグするとして、gcc実行時に``-fopt-info-all=dump``を付けると情報をdumpというファイルに書き出してくれることが分かった。

``reload1.cc``の``spill_failure()``でエラーメッセージが出ており、この関数が呼ばれた時点でもう失敗のようなので呼び出し元をたどると、``find_reload_regs()``の中。

[source,c++]

  for (i = 0; i < n_reloads; i++)
